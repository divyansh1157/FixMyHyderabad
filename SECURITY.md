# Security Policy

## Supported Versions

We actively support and patch the latest version of FixMyHyderabad.

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |
| < 0.1   | No        |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it immediately by contacting the development team directly at security@fixmyhyderabad.org.

Please do not open public issues for security vulnerabilities until they have been addressed. We aim to respond to all reports within 48 hours and provide a fix or mitigation plan within 7 days.
