# Update Project Constitution

## Goal
Update the project constitution to reflect current governance and principles.

## Workflow
1. Identify changes in project goals or technical requirements.
2. Update `.specify/memory/constitution.md`.
3. Propagate changes to templates.
4. Generate Sync Impact Report.

## Generic Guidance
- Ensure principles are declarative.
- Maintain semantic versioning.
- Align with technical reality of the project.
